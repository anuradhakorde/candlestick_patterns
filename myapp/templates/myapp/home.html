About This Project

CandlestickPattern is a Django-based web application designed for analyzing stock market data and detecting various candlestick patterns. The application allows you to:

  • Upload CSV Files: Import historical stock data from various sources
  • Store Data: Efficiently store stock information in PostgreSQL database
  • Pattern Detection: Identify common candlestick patterns like Doji, Engulfing, etc.
  • Visualization: Generate interactive charts using Plotly
  • Analysis Tools: Comprehensive stock market analysis capabilities
